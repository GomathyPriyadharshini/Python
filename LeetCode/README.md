# 🐍 14-Day Python LeetCode Challenge

## 📌 Overview

This repository contains my structured 14-day journey solving curated **Python-focused LeetCode problems** to strengthen Data Structures & Algorithms (DSA) fundamentals.

Each day focuses on a specific topic pattern commonly asked in coding interviews.

## 🎯 Goals

* Build strong problem-solving skills in Python
* Master core DSA patterns (arrays, strings, graphs, etc.)
* Maintain consistent coding discipline
* Prepare for technical interviews
